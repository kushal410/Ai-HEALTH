# Icons

This folder contains UI icons for the application.

## Icon Types
- dashboard.svg - Dashboard icon
- planner.svg - Co-Care Planner icon
- first_aid.svg - First-Aid icon
- medical_history.svg - Medical history icon
- reflection.svg - Co-Reflection icon
- settings.svg - Settings icon
- language.svg - Language switch icon
- theme.svg - Theme switch icon
- help.svg - Help center icon

## Recommended Format
- SVG files preferred (scalable)
- Consistent style and stroke width
- Size: 24x24px or 32x32px viewBox

## Color
Icons should support theming (use currentColor in SVG)
