# Nurse Feelings (Emotions)

This folder contains animated GIFs representing the nurse's 10 emotional states.

## Required Emotions (10 Total)
1. **Happy** - When user completes tasks
2. **Proud** - When user maintains streaks
3. **Worried** - When user misses multiple routines
4. **Sad** - When user is struggling
5. **Energetic** - High energy from user engagement
6. **Angry** - Frustrated (gentle, caring frustration)
7. **Calm** - Peaceful, meditative state
8. **Sleepy** - Low energy, reminding rest
9. **Excited** - Anticipating user's progress
10. **Disappointed** - Mild disappointment (supportive)

## File Naming Convention
- emotion_happy.gif
- emotion_proud.gif
- emotion_worried.gif
- etc.

## Recommended Format
- GIF files with loop
- Size: 150x150px to 300x300px
- Expressive and warm visuals
