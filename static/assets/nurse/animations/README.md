# Nurse Animations

This folder contains animated GIFs or Lottie files for nurse interactions and behaviors.

## Recommended Format
- GIF or Lottie JSON files
- Size: 200x200px to 400x400px
- Loop animations preferred

## Animation Types
- heartbeat.gif - Heartbeat pulse animation
- breathing.gif - Calm breathing animation
- celebration.gif - Success/celebration animation
- thinking.gif - Processing/thinking animation
- waving.gif - Greeting animation

## Usage
These animations enhance the "feels alive" experience by replacing static emojis with dynamic visuals.
