# Sound Assets

This folder contains audio files for enhanced user experience.

## Sound Types
- notification.mp3 - Reminder notifications
- celebration.mp3 - Task completion sound
- heartbeat.mp3 - Heartbeat sound (optional)
- alert.mp3 - Emergency mode alert
- calm.mp3 - Calming background sound

## Recommended Format
- MP3 or OGG format
- Short duration (1-3 seconds)
- Low file size (<100KB)

## Usage
Sounds are optional but enhance the immersive "feels alive" experience.
